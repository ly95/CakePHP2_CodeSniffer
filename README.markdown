About
-----

针对Cakephp2.x项目自定义的代码规范检查，基于 CakePHP_CodeSniffer 1.0 和 PHP_CodeSniffer 1.4.8 。

Requirements
------------

php > 5.3


How
------------

使用：

php /path/CakePHP2_CodeSniffer/scripts/phpcs --standard=CakePHP demo.php
